# Forza Horizon 6 Spotify Mod

Enhance your music experience in Forza Horizon 6 with quick Spotify controls and playlist management.

## Features

- Spotify playback controls
- Playlist switching
- Track skip and previous track
- Volume adjustment
- Lightweight background operation

## Installation

1. Download the latest release.
2. Extract the archive.
3. Run `FH6SpotifyMod.exe`.
4. Launch Forza Horizon 6.

## Requirements

- Windows 10 / 11
- Spotify Desktop Application
