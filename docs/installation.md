# Installation Guide

## FH6 Spotify Mod

1. Download the latest release from [Releases](https://github.com/<your-repo>/releases).
2. Extract the ZIP file.
3. Run `FH6SpotifyMod.exe`.
4. Launch Forza Horizon 6.
5. Enjoy controlling Spotify directly while driving.

## Notes

- Windows 10/11 only
- Spotify Desktop App must be installed
- This is a demo mod; full Spotify API integration requires authorization.
